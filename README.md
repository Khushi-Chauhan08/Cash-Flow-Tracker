## Cash Flow Tracker
A simple web-based Cash Flow Tracker that helps users manage salary, expenses, and remaining balance with visual analytics.

## Live Website Link
https://khushi-chauhan08.github.io/Cash-Flow-Tracker/

## Features
- Add salary and expenses
- Delete expenses
- Automatic balance calculation
- Data saved in browser (LocalStorage)
- Pie chart visualization
- PDF report download

  ## Technology Used
  HTML,CSS,Javascript
  Chart.js, jsPDF
